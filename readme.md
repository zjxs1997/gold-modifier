### Gold-Modifier

写着玩的精灵宝可梦金版（日语版）数据修改器GUI

计划实现的功能有：
- 修改精灵的各种属性值（种族、属性、捕获率、经验值等）
- 修改精灵的技能列表
- 修改精灵的进化链
- 修改初始精灵
- 修改野生精灵分布
- 修改训练师数据
- 修改商店出售的道具
